####

```js
for (var i = 0; i < 10; i++) {
  setTimeout(() => console.log(i)); // 10 만 열번찍힘
}
for (let i = 0; i < 10; i++) {
  setTimeout(() => console.log(i)); // 0 ~ 9 찍힘
}
```

- 페이지 로딩시에 페이지마다 setTimeout()이 호출되는 횟수가 누적
- 구글 메인 페이지에서 실행시 구현상 setTimeout()이 호출되어 setTimeout()이 반환하는 함수 호출 수가 예상한 값이랑 다른듯함
